# ブランチ運用（暫定）

devlopからブランチを切る。ブランチ名は自由だが担当者名をいれること<br>
↓<br>
ローカルで開発<br>
↓<br>
自分が作ったブランチをリモートにプッシュ<br>
※devlopブランチ、masterブランチにはpushしない事<br>
↓<br>
devlopブランチをターゲットにpull requestsを出す<br>
↓<br>
レビュー担当者がレビュー<br>
<br>
<br>
※マージはレビュー担当者が行います<br>
<br>
<br>
