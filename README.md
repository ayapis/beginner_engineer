# ブランチ運用（暫定）

devlopからブランチを切る。ブランチ名は自由<br>
↓<br>
ローカルで開発<br>
↓<br>
リモートブランチにプッシュ<br>
↓<br>
devlopブランチをターゲットにpull requestsを出す<br>
↓<br>
誰かがレビュー<br>
↓<br>
レビュー戻りがなければdevlopにマージ<br>
<br>
